# 二 日历用的是JQ和原生JS相结合来实现的
     思路是对new Date对象以及appendTo()的灵活使用
     
    # 1.知道所要展示的月份一共有多少天
   
    # 2.该月份的1号是周几，这样你就知道再1号之前填补多少空白
   
    # 3.其次该月份一共有多少周，这样是用来计算每块的高度
   
    # 4.这样你就可以根据不同需求，来在每天上添加不同的样式
